## Spaceship Titanic Kaggle Challenge
This repository contains code for "Spaceship Titanic" Challenge on Kaggle. 

Link: https://www.kaggle.com/competitions/spaceship-titanic/data

Authors: @paulaleoni, @LCruzFer
### Goal
The goal is to use the train dataset to build a prediction model that determines whether a passenger on the "space titanic" is transported to another dimension or not. Evaluation is done on a separately provided dataset (test.csv).
